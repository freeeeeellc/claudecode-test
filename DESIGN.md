# DESIGN.md — クリアベル ハウスクリーニングLP デザイン定義書

参考LP（r-cleaning.com）の構成を超える独自LPを、**白ベース・大きめタイポグラフィ**で実装するための設計書。
実装は本書のセクション順に1セクションずつ行う（Claude Codeでの段階実行用）。

---

## 1. デザインコンセプト

> **「清潔は、白で語る。」**
> クリーニングサービスの提供価値＝清潔そのものをデザインで体現する。
> 装飾で売らず、余白・タイポグラフィ・本物の写真で信頼を作る。

- 参考LPとの差別化：赤×チラシ的密度 → **白×編集デザイン的余白**
- 訴求軸：割引額ではなく「明朗会計 × 分解洗浄の品質 × 保証」

## 2. 美学プロンプト（Claude Cookbook 準拠・本プロジェクト適用版）

実装時は常に以下の原則をプロンプトとして適用する：

```
あなたは一流のエディトリアルWebデザイナーです。以下の美学原則を厳守してください。
1. タイポグラフィが主役：明確な階層（display/heading/body/caption）を作り、
   サイズ差は控えめな要素を足すのではなく大胆なスケール比（1.25〜1.5）で表現する。
2. 余白は機能：セクション間 96px 以上。詰めて埋めず、削って語る。
3. 色は禁欲的に：ベース1色（白）＋インク1色＋アクセント1色＋CTA1色。それ以上増やさない。
4. 角丸・影・罫線は全コンポーネントで統一トークンを使う。場当たりで変えない。
5. モーションは控えめに：出現時のフェードアップ（0.6s, ease-out）のみ。装飾アニメ禁止。
6. 写真は明るくトーンの揃ったものだけを使い、同一の角丸・比率で統一する。
7. 文字は大きく：本文17px以上、行間1.9。高齢層でも読めるサイズを下限とする。
```

## 4. タイポグラフィ（文字大きめ設定）

- フォント：**Noto Sans JP**（400 / 500 / 700 / 900）Google Fonts
- ベース：**18px** / 行間 1.9（SP: 16.5px）
- スケール：
  - Display（ヒーローH1）: clamp(34px, 5.4vw, 56px) / 900
  - H2（セクション見出し）: clamp(28px, 4vw, 40px) / 900
  - H3（カード見出し）: 20〜22px / 700
  - 本文: 18px / 400
  - キャプション・注記: 14px
- 数字（価格・実績）は 900 ウェイトで大きく、単位は小さく

## 6. セクション構成と実装指示（この順に1つずつ実装）

1. **ヘッダー**：白・blur固定。ロゴ＋電話＋CTAボタン1つ
2. **ヒーロー**：白背景。左に大型コピー（追加料金ゼロ訴求）＋実績チップ4つ（作業実績・口コミ評価・口コミ件数・保険）、右に窓拭き写真。下にキャンペーン帯（最大◯%OFF）
3. **悩み共起**：チェックリスト6項目 → 解決宣言
4. **選ばれる理由6つ**：snow背景、白カード3×2
5. **実例（Before/After）**：写真3点、フィルタで対比
6. **料金メニュー10種**：二重価格（通常価格打ち消し＋キャンペーン価格＋%OFFバッジ）。参考LPの価格体系に対応
7. **時間制パック4種**：60分／3時間／4時間（人気）／6時間。参考LPのパックプランに対応
8. **空室クリーニングバナー**：写真＋濃色グラデーションの全幅バナー
9. **比較表**：自社列を accent-pale でハイライト
10. **ご利用の流れ4ステップ**：番号を大きく
11. **3つの約束（保証）**：手袋写真＋カード
12. **お客様の声3件**：ダミー明記
13. **FAQ6問**：アコーディオン
14. **お知らせ**：日付＋タグ＋見出しのリスト（参考LPのニュースに対応）
15. **対応エリア**：1都3県チップ＋近隣出張料の注記
16. **注意事項**：キャンセルポリシー・保証期間・駐車スペース等（参考LPの利用規約に対応）
17. **最終CTA**：リビング写真に白オーバーレイ、電話番号特大
18. **フッター＋SP固定CTAバー**

> 二重価格表示（通常価格→割引価格）は、通常価格での販売実績がない場合は有利誤認（景品表示法）となる。公開前に必ず実績ある価格に差し替えること。
